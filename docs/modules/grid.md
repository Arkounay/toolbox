# Grid 

A flexbox grid system with some helper classes for `alignment`, `space distribution` ...

## Base 
<br/>
<Grid-base></Grid-base>

```html
<div class="grid grid-collapse">
	<div class="col-s-7">
		<div>...</div>
	</div>
	<div class="col-s-5">
		<div>...</div>
	</div>
</div>
```

<br/>


## Nested
<br/>
<Grid-nested></Grid-nested>

```html
<div class="grid grid-collapse">
	<div class="col-s-12">
		<div>
			<div class="grid grid-collapse">
				<div class="col-s-7">
					<div>...</div>
				</div>
				<div class="col-s-5">
					<div>...</div>
				</div>
			</div>
		</div>
	</div>
</div>
```
