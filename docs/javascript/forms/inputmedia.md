#  InputMedia.js

## Config
```js
{
	element: document.querySelectorAll('.input-media'), //Element of input parent container
	drag: true //Default value. Add possiblity to drag and drop image in input file
}
```

## Basic

```js
new InputMedia({
	element: document.querySelectorAll('.input-media')
});
```
